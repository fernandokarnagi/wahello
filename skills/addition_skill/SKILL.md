# Addition Skill

This skill performs addition of two numbers.

## Usage
```
add 5 3
```

## Examples
- Add 10 and 20 to get 30
- Calculate 15 + 5 = 20

## Guidelines
- The skill expects exactly two numeric arguments
- Returns the sum of the two numbers
- Handles both integers and floating point numbers
